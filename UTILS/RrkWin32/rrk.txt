                           ReadRKWin v 1.50
               Утилита чтения файлов в формате РК с МЛ.

     1. Назначение.

     Программа предназначена  для  чтения  файлов  на магнитной ленте,
сделанных на компьютерах "Радио-86РК",  "Партнер", "Апогей", "Микроша"
и других, использующих тот же формат записи. Программа не предназначе-
на для анализа и какого-либо преобразования этих файлов и записывает в
файл простую  последовательность считанных байт за исключением ракорда
и первого  синхробайта  (это  формат  .rk,  используемый  в  эмуляторе
emu80) или  с  первым  синхробайтом  (испльзуется в эмуляторе pkemul02
Александра Демина). В случае необходимости преобразования файлов можно
воспользоваться программой ASD-80, опубликованной в журнале "Радио" #7
1990 г.

     Поддерживаемые форматы : RK (эмулятор emu80), PKI, GAM, BSM, BSS,
EDM  (используются  в  эмуляторе  pkemul02 Александра Демина - для них
слежует указать ключ /s).

     2. Системные требования и возможные проблемы :

        - процессор 486 и выше (минимальная частота не определялась)
        - Windows 95/98 или (предпочтительнее) Windows NT
        - любая звуковая карта,  поддерживающая частоту  дискретизации
          44100 Гц моно.

     Программа представляет  собой консольное Win32-приложение и рабо-
тает в текстовом режиме.  Программа может работать в ОС Windows  95  и
NT, Win32s  для  Windows  3.1x  консольные приложения не поддерживает.
Поскольку поддержка консольных приложений в Windows 95 реализована  из
рук вон плохо, возможна нечеткость при вводе с клавиатуры (претензии -
в Microsoft).  Кроме того,  возможны ошибки чтения при фоновой  работе
других приложений, особенно 16-разрядных. В Windows NT никаких проблем
не возникает.


     3. Работа с программой.

     При запуске программы можно задать следующие ключи:

     -d - Используйте для чтения кассет  плохого  качества.  При  этом
          происходит дифференцирование входного сигнала и более устой-
          чивое считывание информации.  Недостаток  -  в  этом  режиме
          программа очень чувствительна к перегрузкам на входе.

     -s - Используйте при необходимости записать файл в формате с пер-
          вым синхробайтом.

     Перед запуском программы убедитесь,  что в микшере  задан  верный
источник  звука  для записи (Line-In для линейного входа звуковой пла-
ты). Положение регулятора уровня записи в  нормальном  режиме  особого
значения не имеет,  но если будут проблемы при считывании,  попробуйте
увеличить уровень.  В режиме с дифференцированием  следует  установить
регулятор приблизительно на уровне 40-50% от максимального.

     После запуска  программы необходимо задать константу чтения в де-
сятичном виде. Нормальное ее значение для используемой в "Радио-86РК",
"Партнере"  и  "Апогее"  скорости 1100 бит/с составляет приблизительно
30.  (Обратите внимание, что оно не связано с константами, задаваемыми
в  директиве I данных компьютеров).  Для "Микроши" нормальное значение
константы примерно 40-45.

     После задания скорости необходимо подготовить магнитофон и нажать
на любую клавишу.  После считывания файла следует нажать Enter для за-
писи файла на диск, ESC - для выхода без записи и любую другую клавишу
для  повторного считывания.  (Чтение можно прервать,  просто остановив
магнитофон.) В случае крайней необходимости можно выйти из  программы,
нажав Ctrl-Break. После записи файла рекомендуется сразу же попытаться
загрузить несоответствия контрольной суммы повторить считывание.


     4. Будущие версии.

     В будущих версиях планируется сделать следующее:

        - возможность автоматического определения константы чтения
        - автоматический  режим  работы  для  считывания  подряд  всех
          встреченных файлов на ленте
        - сделать версию программы для DOS, работающую с
          Sound Blaster Pro - совместимыми звуковыми картами.
