                            Rss2Bin v.1.0
                Утилита для создания файлов снапшотов

     1. Назначение.

     Программа предназначена для создания файлов снапшотов из двоичных
с указанием параметров размещения их в адресном пространстве и старто-
вого  адреса.  Она  может быть применена для загрузки пользовательских
ПЗУ.

     2. Системные требования и возможные проблемы :

        - процессор 8088 и выше
        - DOS

     3. Работа с программой.

     Работа с программой производится в интерактивном режиме. Програм-
ма не принимает никакие аргументы в командной  строке.  После  запуска
программа выдает  ряд запросов,  на которые нужно ввести требуемую ин-
формацию:


Имя rss-файла [.rss]:

     Необходимо ввести  имя результирующего файла.  Если расширение не
указано, то подразумевается .rss


Тип компьютера (1:РК-86 2:Партнер 3:Микроша 4:Апогей 5:Специалист :

     Необходимо выбрать тип компьютера, для которого формируется файл.


Адрес запуска (hex): f800

     Адрес, по которому будет передано управление сразу после загрузки
файла. Обычно  он  должен  быть равен адресу холодного старта Монитора
(F800 или C000 (для "Специалиста")).


Количество блоков данных: 1

     Результирующий файл может иметь несколько блоков данных, размеща-
емых в разных участках адресного пространства.  Обычно Вам не  понадо-
бится использовать более одного блока.

     Дальнейшие параметры заносятся отдельно для каждого блока:


Имя bin-файла с блоком #1 [.bin]:

     Имя двоичного файла, который будет размещен в адресном пространс-
тве эмулируемого ПК. Если расширение не указано, подразумевается .bin


Размер файла XXXX

     Данное значение выводится для справки.


Начальный адрес блока (hex): f000

     Адрес, начиная с которого будет размещен двоичный файл в адресном
пространстве.
