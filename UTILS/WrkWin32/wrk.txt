                          WriteRKWin v 1.10
              Утилита записи файлов в формате РК на МЛ.

     1. Назначение.

     Программа предназначена  для записи файлов на магнитную ленту для
чтения их на компьютерах "Радио-86РК",  "Партнер", "Апогей", "Микроша"
и других,  использующих тот же формат записи.  Программа записывает на
ленту ракорд, синхробайт 0E6H (при необходимости) и последовательность
байт из  указанного файла (Это соответствует форматам,  используемым в
эмуляторах emu80 и pkemul02).


     2. Системные требования и возможные проблемы :

        - процессор 486 и выше (минимальная частота не определялась)
        - Windows 95/98 или (предпочтительнее) Windows NT
        - любая звуковая карта,  поддерживающая частоту  дискретизации
          44100 Гц моно.

     Программа представляет  собой консольное Win32-приложение и рабо-
тает в текстовом режиме.  Программа может работать в ОС Windows  95  и
NT, Win32s  для  Windows  3.1x  консольные приложения не поддерживает.
Поскольку поддержка консольных приложений в Windows 95 реализована  из
рук вон плохо, возможна нечеткость при вводе с клавиатуры (претензии -
в Microsoft).  Кроме того,  возможны ошибки чтения при фоновой  работе
других приложений, особенно 16-разрядных. В Windows NT никаких проблем
не возникает.

     Поддерживаемые форматы : RK (эмулятор emu80), PKI, GAM, BSM, BSS,
EDM (используются в эмуляторе pkemul02 Александра Демина).

     3. Работа с программой.

     Программа принимает в командной строке  единственный  параметр  -
имя файла,  подлежащего выводу на магнитофон.  Формат файла (наличие в
нем первого синхробайта) определяется автоматически.

     После запуска  программы необходимо задать константу чтения в де-
сятичном виде. Нормальное ее значение для используемой в "Радио-86РК",
"Партнере"  и  "Апогее"  скорости 1100 бит/с составляет приблизительно
20.  (Обратите внимание, что оно не связано с константами, задаваемыми
в  директиве I данных компьютеров).  Для "Микроши" нормальное значение
константы примерно 25-30.  (Обратите внимание,  что значение константы
записи  должно  быть примерно в полтора раза меньше значения константы
чтения).

     После задания скорости необходимо подготовить магнитофон и нажать
на  любую  клавишу.  После записи файла на экран выдается сообщение об
успешной записи и происходит выход из программы.  В случае крайней не-
обходимости можно выйти из программы, нажав Ctrl-Break.


     4. Будущие версии.

     В будущих версиях планируется сделать следующее:

        - автоматический режим работы для записи подряд файлов, задан-
          ных списком или маской
        - сделать версию программы для DOS, работающую с
          Sound Blaster Pro - совместимыми звуковыми картами.
